# 同一テキスト複数出現テスト

このファイルは、同じ単語が複数回出現する場合の注釈機能をテストするためのものです。

## テストケース1: 単純な繰り返し

テストという単語が複数回出現します。

- 最初のテスト
- 2番目のテスト
- 3番目のテスト

上記の各「テスト」に別々の注釈を付けられるかを確認してください。

## テストケース2: 文中での繰り返し

このテストは重要です。テストを実行する前に、テストの準備をしてください。

## テストケース3: 表内での繰り返し

| 項目 | 説明 |
|------|------|
| テスト1 | 最初のテスト項目 |
| テスト2 | 2番目のテスト項目 |
| テスト3 | 3番目のテスト項目 |

## テストケース4: コードブロック内での繰り返し

```javascript
function test1() {
  console.log("test");
}

function test2() {
  console.log("test");
}

function test3() {
  console.log("test");
}
```

## テストケース5: 見出しでの繰り返し

### サンプル見出し

本文にサンプルという単語があります。

### 別のサンプル見出し

ここにもサンプルがあります。

## 期待される動作

1. 各「テスト」や「サンプル」に個別に注釈を付けられること
2. 注釈は正しい位置にハイライトされること
3. 注釈をクリックすると正しい箇所にジャンプすること
